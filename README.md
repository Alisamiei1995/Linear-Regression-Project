# Car Price Prediction - Linear Regression
# Project Overview: 
A complete machine learning pipeline predicting used car prices using Linear Regression
## Business Impact:
** For Buyers: Avoid overpaying for used cars
** For sellers: price vehicles competitively
** For Dealers: optimize inventory pricing strategies
# Data Science Pipeline:
** Data Loading & inspection**: cardata.csv
** Data Cleaning**: Handling missing values
** Exploratory Data Analysis**:Visualizing correlations and patterns
** Model Training**: Linear Regression with scikit-learn
** Model Evaluation**: R2 scoe, MAE, RMSE metrics
# skills Demonstrated
** Programming**: python(pandas,numpy,scikit-learn)
** Visualization**: Matplotlib, seaborn
